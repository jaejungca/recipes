Module 2 — Data Splitting & Preprocessing

Cal Poly Pomona · Applied Machine Learning

Published

June 27, 2026

This module covers the foundations of preparing marketing data for machine learning: understanding the probabilistic tools behind classification models, splitting data in a principled way, building reusable preprocessing pipelines with recipes, and verifying that each transformation does what you intend. Work through the four pages below in the suggested order.

Suggested order: Start with the Video Quiz to build conceptual foundations → review the Slides for a big-picture map → work through the Codebook for hands-on R practice → complete the Log Assignments to solidify the mathematics.

Start here · Conceptual foundations

Video Quiz

Test your understanding of four StatQuest videos that build the mathematical foundation for logistic regression — from logarithms to odds to classification models.

Key concepts & skills
  • Logarithms — isolating exponents, base conversion, geometric mean
  • Odds vs. probability — converting between the two
  • Log(odds) / logit — symmetry and the real line
  • Odds ratios — effect size and direction
  • Significance tests — Fisher's exact, chi-square, Wald
  • Logistic vs. linear regression — S-curve, maximum likelihood
  • R: log(), exp(), plogis(), fisher.test(), chisq.test()

Codebook

The main hands-on notebook. Work through the direct-mail dataset from raw data to a fitted Lasso logistic regression, with nine exercises covering common pitfalls and extensions.

Key concepts & skills
  • Log-normal income — understanding right skew visually
  • initial_split() with stratification
  • recipe() — declaring a preprocessing plan
  • step_impute_median(), step_log(), step_normalize()
  • step_dummy(), step_zv(), update_role()
  • prep() / bake() — learn vs. apply
  • workflow(), fit(), augment()
  • Data leakage — why prepping on test data is wrong
  • Variable roles — why ID columns must not be predictors

Step 4 · Solidify the mathematics

Log Assignments

Five focused assignments that verify your ability to perform log transformations in R — converting between dollar and log scale, back-transforming medians and means, and understanding why base choice is irrelevant after normalization.

Key concepts & skills
  • Dollar → log: log() and log10()
  • Log → dollar: exp() and 10^x
  • Log-normal median vs. mean — the σ²/2 correction
  • Geometric standard deviation — multiply/divide, not add/subtract
  • Empirical vs. theoretical values in real data
  • Base 10 vs. natural log — constant ratio, same result after step_normalize()
  • R: mutate(), summarise(), tibble()
